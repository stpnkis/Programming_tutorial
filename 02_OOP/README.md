# 02 - Objektově Orientované Programování (OOP)

Klíčový koncept pro čistý, znovupoužitelný a škálovatelný kód.

## Témata
| # | Složka | Co se naučíš |
|---|--------|-------------|
| 01 | `tridy_a_objekty` | class, __init__, self, atributy, metody |
| 02 | `dedicnost` | dědičnost, super(), MRO |
| 03 | `polymorfismus` | přepisování metod, duck typing |
| 04 | `zapouzdreni` | public/private, properties, gettery/settery |
| 05 | `abstraktni_tridy` | ABC, abstractmethod, interfaces |
| 06 | `magic_methods` | __str__, __repr__, __eq__, __lt__, __add__ |
| 07 | `dataclasses` | @dataclass, field(), frozen, post_init |
| 08 | `composition_vs_inheritance` | "has-a" vs "is-a", kdy co použít |
| 09 | `SOLID_principy` | SRP, OCP, LSP, ISP, DIP |
| 10 | `design_patterns` | Factory, Singleton, Observer, Strategy, State |

## Cíl
Umět navrhnout systém jako sadu spolupracujících objektů. Rozumět kdy použít dědičnost a kdy kompozici.
