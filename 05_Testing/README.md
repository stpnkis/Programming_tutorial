# 05 - Testing

Psát kód, který se dá ověřit. Profesionální standard.

## Témata
| # | Složka | Co se naučíš |
|---|--------|-------------|
| 01 | `unittest` | TestCase, setUp, assertions, test runner |
| 02 | `pytest` | fixtures, parametrize, markers, conftest |
| 03 | `mocking` | Mock, patch, side_effect, MagicMock |
| 04 | `TDD` | Test-Driven Development cyklus: Red → Green → Refactor |
| 05 | `integration_testy` | testování celých pipeline, end-to-end |

## Cíl
Mít návyk psát testy ke každému kódu. Důvěřovat svému kódu.
