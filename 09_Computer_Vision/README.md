# 09 - Computer Vision

Zpracování obrazu — od pixelů k porozumění scéně.

## Témata
| # | Složka | Co se naučíš |
|---|--------|-------------|
| 01 | `opencv_zaklady` | imread, imshow, resize, crop, color spaces |
| 02 | `image_processing` | filtry, blur, threshold, morphologie |
| 03 | `detekce_hran_a_features` | Canny, Harris, ORB, SIFT, feature matching |
| 04 | `object_detection` | YOLO, SSD, Haar cascades, bounding boxes |
| 05 | `segmentace` | semantic/instance segmentation, watershed, U-Net |
| 06 | `camera_calibration` | intrinsics, distortion, checkerboard, stereo |
| 07 | `depth_estimation` | stereo vision, disparity maps, depth cameras |
| 08 | `point_clouds` | Open3D, PCL basics, 3D vizualizace |
| 09 | `cv_pipeline_projekt` | end-to-end CV pipeline (capture → detect → act) |

## Cíl
Umět zpracovat obraz od raw snímku po detekci objektů a 3D rekonstrukci.
