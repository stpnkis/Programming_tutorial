# 12 - Networking a API

Komunikace mezi systémy — zásadní pro robotiku i ML.

## Témata
| # | Složka | Co se naučíš |
|---|--------|-------------|
| 01 | `http_requests` | requests library, GET/POST, JSON, headers |
| 02 | `REST_API_flask` | Flask/FastAPI, endpointy, CRUD, response codes |
| 03 | `websockets` | real-time komunikace, asyncio websockets |
| 04 | `serial_komunikace` | pyserial, UART, Arduino/MCU komunikace |
| 05 | `mqtt` | publish/subscribe, mosquitto, IoT messaging |

## Cíl
Umět propojit libovolné systémy — robot↔server, sensor↔PC, API↔frontend.
