# 03 - Datové Struktury a Algoritmy

Srdce programátorského myšlení. Umět vybrat správnou strukturu = rychlejší kód.

## Témata
| # | Složka | Co se naučíš |
|---|--------|-------------|
| 01 | `arrays_a_linked_lists` | list vs deque, indexování, paměťový model |
| 02 | `stacks_a_queues` | LIFO/FIFO, implementace, use cases |
| 03 | `stromy_a_grafy` | binary tree, BFS, DFS, adjacency list |
| 04 | `hashovani` | hash tables, dict internals, collision |
| 05 | `sorting_algoritmy` | bubble, merge, quick sort, sorted() |
| 06 | `searching_algoritmy` | linear, binary search, bisect |
| 07 | `rekurze` | base case, stack overflow, tail recursion |
| 08 | `dynamicke_programovani` | memoization, tabulation, klasické úlohy |
| 09 | `Big_O_analyza` | O(1), O(n), O(log n), O(n²), space complexity |
| 10 | `prakticke_ulohy` | LeetCode-style problémy, coding challenges |

## Cíl
Rozumět složitosti algoritmů a umět vybrat efektivní řešení.
